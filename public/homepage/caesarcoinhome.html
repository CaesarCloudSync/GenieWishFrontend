CaesarCoin BlockChain

DISCLAIMER: When sending requests to these API's the server will take about 10 seconds to warm up.

This blockchains allows you to create transactions and get your wallet balance Next I plan to write the proof of work code locally in C++ using SHA256 hashing.

The blockchain is not yet persistent and will only stay on runtime. This API also allows the user to upload and fetch torrent files to the CaesarCoin Database.

Caesar Mine

TO mine a coin:

Send POST request to https://caesarcoinblockchain.onrender.com/mine_block

JSON: {"miner":"{name}"}

Caesar Transaction

TO make a transaction to another user:

Send POST to https://caesarcoinblockchain.onrender.com/make_transaction

JSON: {"sender":"{name}","recipient":"{name}","amount":5}

Caesar Get BlockChain

TO fetch the blockchain:

Send GET to https://caesarcoinblockchain.onrender.com/chain

Caesar Fetch Balance

TO fetch the balance of a user:

Send POST to https://caesarcoinblockchain.onrender.com/get_balance

JSON: {"user":"{name}"}

Caeasar Upload Torrent File

To upload a torrent file:

Send POST to https://caesarcoinblockchain.onrender.com/upload_torrent_file

FormData: file with key as "torrentfile" and file as value

Caeasar Download Torrent File

To download a stored torrent file:

Send POST to https://caesarcoinblockchain.onrender.com/download_torrent_file

JSON {"torrentfile":"{filename}"}
